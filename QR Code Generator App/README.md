# QR Code Generator App

A simple Python application to generate QR codes for given comma-separated words, built using Tkinter for the GUI.

## Features
- Input comma-separated words
- Generate separate QR codes for each word
- Display generated QR codes in the GUI
- Download QR codes as PNG images

## Installation

1. Clone the repository:
   git clone https://github.com/yourusername/QRCodeGeneratorApp.git
   cd QRCodeGeneratorApp
2. Install the required libraries:
   pip install -r requirements.txt

3. Run the application:
   python main.py
