# File Transfer App

A simple Flask web application for transferring images and files between a mobile phone and a PC.

## Features

- Upload files from your mobile phone or PC.
- View the list of uploaded files.
- Download files to your mobile phone or PC.

## Requirements

- Python 3.x
- Flask

## Installation

1. Clone the repository or download the source code.

    ```sh
    git clone https://github.com/yourusername/file-transfer-app.git
    cd file-transfer-app
    ```

2. Install Flask.

    ```sh
    pip install Flask
    ```

3. Run the application.

    ```sh
    python app.py
    ```

## Usage

1. Open your browser and navigate to `http://<your_pc_ip>:5000`.
2. Use the "Upload File" form to upload files.
3. The "Uploaded Files" section displays the list of uploaded files. Click on any file to view or download it.
4. Use the "Download File" form to download files by entering the filename.
